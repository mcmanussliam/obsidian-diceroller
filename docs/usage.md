# User Guide

Obsidian Dice Roller lets you roll dice directly from Obsidian and shows the
result after the dice settle.

## Open The Roll Modal

1. Open the command palette.
2. Run **Roll dice**.
3. Enter dice notation.
4. Press Enter or click **Roll**.

Common examples:

```text
1d20
2d6+3
4d6kh3
1d20+STR+prof
```

The modal also includes quick buttons for common dice.

## Roll Selected Text

You can roll notation directly from a note.

1. Select dice notation in the editor.
2. Right-click the selection.
3. Choose **Roll dice**.

Example:

```markdown
**War Pick.** `1d20+STR+prof` to hit. Hit: `1d8+STR` piercing damage.
```

## Quick Roll Commands

The plugin adds commands for individual dice:

```text
Roll d4
Roll d6
Roll d8
Roll d10
Roll d12
Roll d20
Roll d100
```

These work well with Obsidian hotkeys.
